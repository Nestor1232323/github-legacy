# Тестовый Markdown-файл

Это демонстрационный файл для проверки возможностей парсера Markdown.

---

## Заголовки (Headings)

## Форматирование Текста (Text Formatting)

В этом разделе показано строковое форматирование.

* Обычный абзац текста.
* Здесь мы используем **жирный текст** (`**text**`) для выделения.
* Также мы можем использовать *курсив* (`*text*`) для акцента.
* Можно комбинировать: **жирный текст с *курсивом***.

### Встроенный Код (Inline Code)

Для коротких фрагментов кода используется встроенный код, например, метод `renderMarkdownLine:` или переменная `insideCodeBlock`.

---

## Списки (Lists)

Поддерживаются маркированные списки с использованием `-` или `*`.

* Первый пункт списка
* Второй пункт, который может быть **жирным** или *курсивным*.
- Третий пункт, демонстрирующий использование дефиса.

---

## Блоки Кода (Code Blocks)

Парсер корректно обрабатывает блоки кода, заключенные в тройные обратные кавычки (```).

```objective-c
// Пример кода Objective-C
- (void)someMethod {
    NSString *test = @"Hello, World!";
    // Блок кода должен быть моноширинным
    // и иметь специальный фон и шрифт.
    NSLog(@"Test: %@", test);
    
    // Проверка многострочности
    if ([test length] > 0) {
        return;
    }
}
